# Cloudflare Workers 検証用コード: R2 へのファイルアップロード

このプロジェクトは、Cloudflare Workers の動作検証を目的とした**サンプルコード**です。
特に R2 ストレージとの連携およびパフォーマンス特性を検証するための実験的な実装となっています。

## ⚠️ 検証用コードの注意事項

- このコードは**実験・検証目的**で作成されています
- 本番環境での使用は想定されていません
- セキュリティや最適化の観点から不十分な実装を含みます
- Cloudflare の仕様変更により動作が変わる可能性があります

## 検証内容

このサンプルコードでは以下の検証が可能です：

- Cloudflare R2 からのファイルアップロード処理
- Workers 環境でのリクエスト制限の挙動確認

## 技術スタック

- Hono（Web フレームワーク）
- Cloudflare Workers（サーバーレス実行環境）
- Cloudflare R2（オブジェクトストレージ）
- TypeScript
